# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Commands

```bash
# Set up the SQLite database with sample patient data
python setup_db.py

# Run the A1C decision support script
python patients.py

# Create today's dated folder on the Desktop
bash myday.sh
```

## Architecture

This is a small learning/exploration repository with a few independent scripts:

- **`patients.py`** — Clinical decision support script. Reads patient records from `patients.db` (SQLite) and prints follow-up recommendations based on A1C thresholds: >9.0 = urgent, >7.5 = routine, ≤7.5 = at goal.
- **`setup_db.py`** — One-time script to create and seed `patients.db` with a `patients` table (`mrn`, `name`, `age`, `a1c`). Run this before `patients.py` if the database doesn't exist.
- **`myday.sh`** — Utility script that creates a dated folder (`YYYY-MM-DD`) on the Desktop with an empty `notes.txt` inside.
- **`patients.db`** — SQLite database file (generated by `setup_db.py`; not source-controlled data).
